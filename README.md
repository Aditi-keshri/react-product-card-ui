# React Product Card UI

## Project Overview

This project is a responsive product card UI built using React and CSS. It simulates an e-commerce product listing interface with reusable components, simple authentication UI, and a clean, modern layout.

## Objective

To demonstrate frontend development skills including component-based architecture, responsive design, CSS grid layout, and interactive UI styling.

## Features

- Responsive product grid layout
- Reusable `ProductCard` component
- Product image, title, price, and Add to cart button
- Login and signup form
- User profile display after login or signup
- Product page access restricted until the user is logged in
- Hover effects on product cards, images, and buttons
- Mobile-friendly design

## Tech Stack

- React
- Vite
- CSS3
- CSS Grid
- Flexbox

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

### 3. Open in your browser

```text
http://127.0.0.1:5173/
```

### 4. Build for production

```bash
npm run build
```
