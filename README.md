# React Product Card UI

## Project Overview

This project is a responsive product card UI built using React and CSS. It simulates an e-commerce product listing interface with reusable components and a clean, modern layout.

## Objective

To demonstrate frontend development skills including component-based architecture, responsive design, CSS grid layout, and interactive UI styling.

## Features

- Responsive product grid layout
- Reusable `ProductCard` component
- Product image, title, price, and Add to cart button
- Hover effects on product cards, images, and buttons
- Mobile-friendly design

## Tech Stack

- React
- Vite
- CSS3
- CSS Grid
- Flexbox

### 4. Build for production


npm run build
